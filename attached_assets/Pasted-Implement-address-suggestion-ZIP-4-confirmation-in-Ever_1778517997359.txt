api-server/server.mjs

Required backend:
Create or verify:

  "state": "NJ",
  "zip_code": "07305-1055"
}

{
  "address_validated": true,
  "formatted_address": "11 College Dr, Jersey City, NJ 07305-1055",
  "address_line1": "11 College Dr",
  "city": "Jersey City",
  "state": "NJ",
  "zip_code": "07305",
  "zip_plus4": "1055",
  "postal_code_full": "07305-1055",
  "latitude": null,
  "longitude": null,
  "google_place_id": null,
  "dpv_confirmation": null,
  "address_confidence": "HIGH"
}

Required frontend:
Before creating booking in handleSubmit:

1. Call POST `${API}/address-intelligence`.
2. If formatted_address or postal_code_full is returned, show a confirmation card:
   "We found this standardized address. Please confirm."
3. Show:
   - formatted_address
   - postal_code_full
   - address_confidence

   - Use this address

   - postal_code_full
   - address_validated
   - formatted_address
   - latitude
   - longitude
   - google_place_id
   - dpv_confirmation
   - address_confidence
